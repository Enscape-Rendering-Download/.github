## 🚀 Introduction
Enscape for Windows brings rendering into the design process itself. Instead of exporting a scene and waiting for an image, you keep working in your usual tool while Enscape shows a photorealistic, always-current view alongside it. The result is a tighter loop between designing and seeing: architects and designers can evaluate light, materials, and space in real time, present convincing visuals to clients, and walk through the project in virtual reality — all without leaving the model.

## 💻 System Specs

**Minimum**
- Windows 10 64-bit
- A modern 64-bit multi-core processor
- A dedicated GPU with current drivers and ray-tracing support
- At least the RAM recommended for your host design software
- Several gigabytes of free storage for the plugin and cache

**Recommended**
- Windows 11 64-bit
- A recent high-performance multi-core processor
- A high-end dedicated GPU with ample video memory for smooth rendering
- Extra RAM beyond your design tool's baseline for large scenes
- A fast SSD with room for project files and exports

## 🎯 Getting Up and Running
1. Open a project in your design application.
2. Launch Enscape from the plugin toolbar to open the live render view.
3. Move through the scene with mouse and keyboard to inspect it from any vantage point.
4. Change materials, sky, and light and see the render respond instantly.
5. Export an image or panorama, or enter VR to review the space at scale.
